# Numerics

## Expressions

Creates a new numeric value

### Literal
- Just the number
- Can have leading zeros
- ``1`` or ``0001``

### Addition
- Adds two values

### Subtraction
- Subtracts two values

### Multiplication
- Multiply two values

### Division
- Divide two values

### Exponentiation
- Raise a value to the power of another value
